## Overview
End-to-end data analysis project on 1,067,371 rows of 
real retail transaction data across 2 years and 41 countries.

## Tools Used
- Python (pandas) — data cleaning + analysis
- Power BI — 3-page executive dashboard
- CSV exports — cleaned data for visualization

## Data Cleaning
- Started with 1,067,371 rows
- Removed 261,822 rows of dirty data (24%)
- Removed cancelled orders, returns, missing customers
- Added Revenue column, extracted Month/Year

## Key Business Insights
- £17.7M total revenue · 5,878 customers · 41 countries
- UK drives 82.98% of all revenue
- November is peak month — confirmed across both years
- 267 VIP customers = 50.49% of total revenue
- Top product: REGENCY CAKESTAND 3 TIER at £286K
